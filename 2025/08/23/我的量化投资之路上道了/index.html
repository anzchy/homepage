我的量化投资之路上道了

Created2025-08-23|Updated2025-08-24|编程自学投资
|Post Views:

在大学的时候，我上过一门证券投资分析课程，当时就听课程老师说江恩、道氏理论等各种证券的技术分析流派，还有大学里比较盛行的模拟炒股大赛，当时一心搞学习（其实应该早点接触投资的），也没有好好研究过证券投资分析。

在工作后的第二年，2018 年开始正式进入股市，刚开始也是没有头绪，跟着大 V买股票、基金。逐步了解到，投资界一直存在着两种比较经典的理论流派，一个是技术分析派，一个是基本面派，技术分析派现多存在于散户，譬如上海人民广场附近的炒股大爷们，神叨叨的也是技术分析派，基本面派则是根据上市公司的质量、护城河、估值等综合判断，强调”以好的价格持有优质的公司”。这两个技术流都挺高深的，反正我是觉得离我挺远的。

2020 年左右，我看了持有封基老师的一本书《10 年 10 倍：散户也可以学习的量化投资方法》，其中介绍到老师通过 Excel 表分析，测试了所有的技术分析指标，譬如”双均线”、红三兵”、”两只乌鸦”，我尝试了下，还是觉得挺难的，Excel 表技术不够啊。

2022年通过看视频，想学习 Python 量化，所以花了大几千块报名 Python 量化课，可惜 Python 基础不够，只是对 Python 简单会用，量化对我来说更加深不可测了。

辗转间到了 2024 年，有了 ChatGPT、豆包，我开始学习 Python 了，这些模型对于编程知识的掌握，比很多学校的讲师还强，而且能根据你个性化的问题给出针对性答案，最关键是及时反馈给你。所以我开始学习 Python，半年时间差不多就上手了，能做一些项目。

2024 年，想做一些可转债的回测，可惜知识不够，连一些股票经典的技术分析指标到底有没有效。

到了 2025 年，看到很多自媒体也在做可转债量化，所以我也摩拳擦掌，觉得学习了 Python，应该基础具备了，先是接触了禄得、果仁网，这两个平台适合不会编程的人，只要找到了对的因子组合，就能回测因子是否有效。

前几天我又看了一下刑大的B 站视频，特别是怎么评价技术分析指标的有效性，这张截图给了我启发，一种我也能行的自信感涌上心头。

我感觉 Excel 应该也能完成吧。先尝试了用 Excel 完成，没想到真的实现了。

用 Excel 完成之后，大概知道数据的分析处理流程，于是开始用 Python 实现代码，目的是遍历 A股几千只股票20 年的日线数据，得到如下双均线技术分析指标是否有效的统计结果，经过 2-3 天的摸索，终于成功跑出结果。

自此，我的量化投资学习之路，终于走上了正轨，也就是掌握了工具，跑通了第一个案例，之后会更加有自信地做其他尝试。

下面做个总结，主要涉及到 Excel 做技术分析指标有效性测试和 Python 做技术分析的一些心得。

一、做量化回测的步骤

无论是 Excel，还是 Python，都可以一站式实现数据清洗、数据分析，得出最终的分析报告。但是 Excel 表只能针对单个股票的数据进行分析，以万科股票(sz000002)为例，自上市以来，其日线数据就有 8000-9000 条，对这单张报表数据的分析，我的 Macbook Pro 2024 尚能应付；如果要对 A 股 5738只股票进行一起分析，有两种方法，一种是合并在一张大表中，这显然不现实，那么一张表就有 400 +万行，电脑肯定吃不消。要么就一个个股票单独分析，这也很费时，这时候 Python 做量化分析的优势就显现了。

做技术指标分析有效性测试的步骤：

1.获取数据

好的数据源特别重要，整个 A 股的数据，可以在预测者网下载，当然刑大所在的彩虹集团（刑大所在的公司）有数据客户端，可以直接下载整理好的数据。下载了数据之后，我们要看看是否可以产生复权后的价格，譬如预测者网有”真实涨跌幅”数据，其中对数据有如下批注：

这里的涨跌幅是股票当天的复权之后的真实涨跌幅，因此可以根据真实涨跌幅数据来计算复权因子

而彩虹集团数据客户端下载的数据中，股票的字段有收盘价、开盘价、最高价、最低价、前收盘价都要有，特别注意要有好的数据源，其中前收盘价特别重要，用于产生复权因子，进而获得收盘价_后复权，因为收盘价在派息之后会有变化，

2.清洗数据

通常需要对其中的空白数据行进行清理。彩虹集团客户端下载的数据都清洗过，比较规范。

3.产生信号

根据技术分析指标产生买入或者卖出信号，譬如经典的双均线理论是这样的：

当5 日均线上穿 20 日均线的时候，产生买入信号；当 5 日均线下穿 20 日均线的时候，产生卖出信号。

4.计算策略效果

有两种方法，一种是基于数学的统计方案，即：

买入信号、卖出信号产生之后的 N 个交易日之后的股票相对于买入时点的 N 日收益情况

假设我在 20250822 晚上，根据 20250821 和 20250820 的数据判断产生了买入信号，那么我在 20250823 早盘买入这个股票（假设开盘能买进去），那么我需要去统计 1 天、3 天、5 天、10 天、20 天后的收盘价相对于我买入时价格的涨跌幅。

当然一只股票的数据量不够，以万科为例，20 年时间，产生的买入和卖出信号也就 400 次左右，数据量不够。因此我们要统计 A 股所有股票的信号数量。

还有一种方法更为常用，就是根据这个策略进行操作，然后计算 10 年后（至少 1 个牛熊，也就是至少 5 年）的每年收益，与基准收益率（譬如沪深 300 指数每年的收益率，中证 2000 指数每年收益率等）做对比，看看是累计跑赢还是跑输。

5.对步骤 4 产生的策略与收益情况做统计分析

三、解读双均线统计结果，并延展到其他技术分析指标

对A股5738支股票进行双均线分析（短线是5日均线，长线是20日均线，短线上穿长线是金叉，是买入信号；短线下穿长线是死叉，是卖出信号）的分析结果解读如下：

买入信号数和卖出信号数各有50万个，在卖出信号(signal是-1）中，1日-20日后的上涨概率（yield是正数的个数占比）是50%左右，1-5日是超过50%，10、20日是低于50%。对比我的解读是：如果卖出信号真有效的话，那么win_ratio应该都低于50%才对，实际上产生卖出信号后，短期（1-5日内）反而股票上涨概率更大。

而产生买入信号(signal是1)的情形中，1-5日后上涨的概率反而低于50%，10-20日后上涨的概率略高于50%。
